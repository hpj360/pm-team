## 1. 设计规范

### 1.1 布局规范

#### 1.1.1 页面布局结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              顶部导航栏 (64px)                               │
├─────────────────────────────────────────────────────────────────────────────┤
│         │                                                                   │
│         │                                                                   │
│  侧边栏  │                        主内容区域                                 │
│ (240px) │                      (min-height: calc(100vh - 64px))             │
│         │                                                                   │
│         │                                                                   │
├─────────┴───────────────────────────────────────────────────────────────────┤
│                              底部信息栏 (可选)                                │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 1.1.2 栅格系统

| 参数 | 值 |
|-----|-----|
| 栅格列数 | 24列 |
| 栅格间距 | 16px |
| 容器最大宽度 | 1920px |
| 内容边距 | 24px |

#### 1.1.3 响应式断点

| 断点 | 宽度 | 侧边栏 | 主内容 |
|-----|------|-------|-------|
| xl | >= 1920px | 展开 240px | 流式宽度 |
| lg | 1200px - 1919px | 展开 240px | 流式宽度 |
| md | 992px - 1199px | 收起 64px | 流式宽度 |
| sm | 768px - 991px | 隐藏 | 100% |
| xs | < 768px | 隐藏 | 100% |

#### 2.1.2 组件说明

| 登录失败 | 显示错误提示，错误次数+1 |
| 连续失败5次 | 账户锁定30分钟 |
