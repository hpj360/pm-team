```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              用户登录流程                                    │
└─────────────────────────────────────────────────────────────────────────────┘

开始
  │
  ▼
┌─────────────────┐
│  访问登录页面    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  选择登录方式    │ ◄──────────────────────────────────┐
│  (密码/LDAP/SSO) │                                    │
└────────┬────────┘                                    │
         │                                             │
         ▼                                             │
┌─────────────────┐                                    │
│  输入认证信息    │                                    │
└────────┬────────┘                                    │
         │                                             │
         ▼                                             │
┌─────────────────┐                                    │
│  点击登录按钮    │                                    │
└────────┬────────┘                                    │
         │                                             │
         ▼                                             │
┌─────────────────┐    否    ┌─────────────────┐       │
│  验证通过？      ├────────►│  显示错误提示    │       │
└────────┬────────┘          │  (错误次数+1)   │───────┘
         │ 是                 └─────────────────┘
         ▼
┌─────────────────┐    是    ┌─────────────────┐
│  锁定账户？      ├────────►│  显示锁定提示    │
└────────┬────────┘          │  (等待30分钟)   │
         │ 否                 └─────────────────┘
         ▼
┌─────────────────┐
│  创建会话        │
│  记录登录日志    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  跳转首页        │
└────────┬────────┘
         │
         ▼
结束
```

**流程说明：**

| 步骤 | 操作 | 系统响应 | 异常处理 |
|-----|------|---------|---------|
| 1 | 用户访问登录页面 | 显示登录表单 | - |
| 2 | 选择登录方式 | 切换对应表单 | - |
| 3 | 输入认证信息 | 实时验证格式 | 格式错误即时提示 |
| 4 | 点击登录 | 提交认证请求 | 网络错误提示重试 |
| 5 | 验证通过 | 创建会话，跳转首页 | 验证失败显示错误 |
| 6 | 连续失败5次 | 锁定账户30分钟 | 提示等待解锁 |

**关键交互点：**

|---------|---------|---------|---------|
| 文件选择 | 拖拽或点击选择 | 高亮上传区域 | - |
| 校验阶段 | 等待 | 显示校验进度 | 0-10% |
| 哈希计算 | 等待 | 显示计算进度 | 10-20% |
| 上传阶段 | 可暂停/取消 | 实时进度条 | 20-90% |
| 服务端处理 | 等待 | 显示处理状态 | 90-100% |
| 完成 | 查看/跳转 | 成功提示 | 100% |

## 3. 操作反馈设计

### 3.1 反馈类型

| 拖拽 | 视觉反馈 | 高亮放置区域 |

| 操作类型 | 确认方式 | 示例 |

## 4. 异常处理流程

### 4.1 网络异常处理

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                            网络异常处理流程                                  │
└─────────────────────────────────────────────────────────────────────────────┘

请求发送
  │
  ▼
┌─────────────────┐
│  网络请求超时？   │
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
   是        否
    │         │
    ▼         ▼
┌───────────┐ ┌───────────┐
│ 显示超时   │ │ 处理响应   │
│ 提示      │ └─────┬─────┘
└─────┬─────┘       │
      │             │
      ▼             ▼
┌───────────┐ ┌───────────┐
│ 重试/取消  │ │ 业务处理   │
└───────────┘ └───────────┘
```

**网络异常处理策略：**

| 异常类型 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 请求超时 | 自动重试3次 | "网络请求超时，正在重试..." |
| 网络断开 | 显示离线状态 | "网络连接已断开，请检查网络" |
| 服务器错误 | 显示错误页面 | "服务器暂时无法响应，请稍后重试" |
| 请求取消 | 不显示提示 | - |

### 4.2 业务异常处理

#### 4.2.1 文件上传异常

| 上传中断 | 4004 | 保存断点 | "上传已暂停，可随时继续上传" |
| 文件校验失败 | 4005 | 提示重传 | "文件校验失败，请重新上传" |

| 结果过多 | 5003 | 分页显示 | "结果过多，请添加更多筛选条件" |

#### 4.2.3 权限异常处理

| 异常场景 | 错误码 | 处理方式 | 用户提示 |
