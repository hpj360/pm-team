## 附录

### 附录A：页面标识规范

页面标识采用 `PAGE_模块_功能` 的命名规范，全部大写，用下划线分隔。

示例：
- `PAGE_FILE_UPLOAD` - 文件上传页面
- `PAGE_TARGET_PROFILE` - 目标画像页面
- `PAGE_ADMIN_USERS` - 用户管理页面

### 附录B：页面路径规范

页面路径采用小写字母，用连字符分隔单词，支持动态参数。

示例：
- `/files/upload` - 文件上传页面
- `/files/detail/:id` - 文件详情页面（带动态参数）
- `/admin/users` - 用户管理页面
